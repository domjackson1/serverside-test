# serverside-test

Spring Boot Web Scraper application with JUnit and Cucumber tests.

## Run Locally
### Build
```sh
$ ./gradlew build
```

## Run application
```sh
$ ./gradlew bootRun
```

## Run Unit tests
```sh
$ ./gradlew test
```

## Run Cucumber tests
```sh
$ ./gradlew cucumber
```

## Build Docker image
```sh
$ ./gradlew build docker
```
